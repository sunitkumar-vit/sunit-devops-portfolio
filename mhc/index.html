MyHealthClinic (MHC)

Enterprise Healthcare Platform

Project Overview

MyHealthClinic (MHC) is an end-to-end DevOps portfolio project demonstrating Kubernetes, Azure DevOps, Terraform, Docker, HashiCorp Vault, Grafana, Prometheus and Cloudflare integration.

Project Highlights

Multi-Environment Deployment

Development, Pre-Production and Production environments deployed on Kubernetes.

CI/CD Automation

Azure DevOps multi-stage pipelines for build, deployment and infrastructure management.

Infrastructure as Code

Terraform used for Azure resource provisioning and remote state management.

Secrets Management

HashiCorp Vault integrated using AppRole authentication.

Monitoring

Grafana and Prometheus dashboards for cluster and application monitoring.
